# MyFlix Movie API
This project is to be an API for users to see a list of movies and mark movies as favorites.


## Features

- Create new user
- Create token to log a user in
- Get list of all movies
- Return data about a Genre
- Return data about a Director
- Get a single movie
- Allow users to deregister
- Users can add and remove a movie from their favorite list
- User can update their information

## Development

### Install
Run `npm install` from the project directory.

### Running it
To run this locally, run `npm run start`
